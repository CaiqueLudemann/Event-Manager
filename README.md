# CP Dashboard

A full-stack event management system built with **React (Vite)** on the frontend and **Node.js + Express** on the backend.

## Features

- Add and manage clients
- Create and track events
- Define event types and services
- Calculate billing dynamically

## Tech Stack

- Frontend: React + TailwindCSS + Axios
- Backend: Node.js + Express + FS (JSON as mock database)

## How to Run Locally

```bash
# in one terminal
cd CPDashboardBackEnd
npm install
npm run start

# in another terminal
cd CPDashboardFrontEnd
npm install
npm run dev
```
